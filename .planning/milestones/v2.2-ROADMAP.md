## Milestone Summary

**Key Decisions:**
- Default sort: created_at descending (newest first)
- Same columns in both modes for visual consistency
- Drag mode headers visible but not clickable
- Space to enter grabbed state (not immediate arrow movement)
- Cmd/Ctrl modifiers for jump to top/bottom
- Escape reverts to original order captured at grab time
- Derive displayName from email username with title case conversion
- Use colspan=3 for idle table rows to span Order/Company/Status columns

**Issues Resolved:**
- Tab badge visibility (badges now use Badge component with secondary variant)
- Sidebar dropdown positioning (now opens to right with proper offset)
- Idle state display (clean "Idle" text, no em-dashes)
- Duplicate table implementations (unified into single QueueTable)

- None
